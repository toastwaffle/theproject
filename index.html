theproject by toastwaffle

A web services management system

This is a programming project by Samuel Littley and Kevin Varley, primarily to gain experience with programming. Written in python, and using GTK+, it is a system for managing websites and projects, including git integration, ftp system, automatic install of web platforms such as joomla, wordpress etc.
